Verse backgrounds for the visual stage.

FOLDER LAYOUT — one folder per book, one subfolder per chapter:

  visuals/
    default.png              fallback for everything (optional)
    Genesis/
      Genesis.png            whole-book fallback (optional)
      1/
        Genesis_1_1.png      exact verse
        Genesis_1.png        whole-chapter fallback (optional)
      5/
        Genesis_5_3.webp
        Genesis_5_3.KJV.webp translation-specific variant
    1_Samuel/                books with spaces use underscores
      17/
        1_Samuel_17_4.png

Most specific file wins: verse -> chapter -> book -> default.
(Files placed flat in visuals/ still work, but the folders above are
preferred — much easier to navigate.)

FORMATS
  PNG / JPG    photos — scaled to fill the screen, center-cropped
  WebP / GIF   diagrams & animations — letterboxed, never cropped.
               Animated WebP preferred over GIF (smaller, full color).
               Saved with loop=1, an animation plays ONCE then holds
               its last frame; loop=0 loops forever.

TRANSLATION-SPECIFIC FILES
Add the version code before the extension and that file is used only
when that version is being read (falls back to the generic file):
  Genesis_5_3.KJV.webp   only for the KJV
Codes: KJV, WEB, ASV, BBE, DARBY, DRA. Useful when a visual contains
text, e.g. name spellings (KJV "Enos" vs Douay-Rheims "Henoch").

MOBILE VARIANTS (web app only)
Add ".mobile" before the extension and that file is preferred on
phone-sized / touch screens, falling back to the standard file:
  Genesis_5_3.mobile.webp        phone version (bigger text, simpler)
  Genesis_5_3.KJV.mobile.webp    phone + KJV-specific
No need to duplicate every visual — only re-cut the ones whose text is
too small on a phone. Recommended size for mobile stills: 1280 x 720
with text sized as if the image were half that.

BUNDLED SETS (all generated by scripts in the project root)
  Genesis/1      photo stills (sources in CREDITS.txt)
  Genesis/5,11   lifespan-timeline genealogies
  Exodus/25-27   tabernacle diagrams
  Numbers/32     Transjordan map (Sihon & Og, Gad/Reuben cities)
  Numbers/34     borders of Canaan traced + twelve-tribe allotment map
  Matthew/1      genealogy of Jesus (three fourteens)
  Luke/3         genealogy of Jesus, backward to Adam (77 names)

Recommended size for stills: about 1920 x 1080. The verse reference
and translator notes (both toggleable in the app) are drawn along the
bottom of the image.
